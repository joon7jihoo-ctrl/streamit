# 👨‍💻 자기소개 페이지 — Streamlit

Streamlit으로 제작한 개인 자기소개 (포트폴리오) 웹 앱입니다.

## 🚀 실행 방법

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 앱 실행

```bash
streamlit run app.py
```

브라우저에서 **http://localhost:8501** 로 접속하세요.

---

## 📁 파일 구조

```
streamit/
├── app.py              # 메인 Streamlit 앱
├── requirements.txt    # 의존 패키지 목록
└── README.md           # 프로젝트 설명
```

## 🛠️ 기술 스택

- [Streamlit](https://streamlit.io/) `>=1.32`
- [streamlit-extras](https://extras.streamlit.app/) `>=0.4`

## 📄 주요 기능

- 히어로 프로필 섹션 (이름, 직함, 뱃지)
- 핵심 지표 카드 (경력, 프로젝트, 기술스택, 자격증)
- 기술 스킬 프로그레스 바
- 경력 타임라인
- 주요 프로젝트 카드
- 자격증 & 교육 이력
- 메시지 폼 (인터랙티브)
- 다크 글래스모피즘 디자인
